Exercise 2.3

Your sketch:

Instructions

Open exercise2-3/sketch.js, add the setup() and draw() functions and create a canvas of 600px by 600px. Put a rectangle or square somewhere on the canvas. The shape can be any size you like as long as there is room to move it around. Set the background colour to black and the fill colour to white.

Make it possible to drag the shape around the canvas. The mouse button must be held down and the mouse pointer must be over the shape when the mouse button is pressed. While the shape is being dragged, change its fill colour (to any colour). As soon as the dragging stops, change the fill colour back to white.

When you are done, check your work using check-exercise2-3.html.

Expected output:

Below are some suggestions. Feel free to try a different approach.

  • Create a global variable called shapeSelected that will store a boolean indicating whether the mouse was over the shape when the mouse button was pressed. Note that the calculation for determining if the cursor is over the shape will look slightly different if you have set rectMode() to CENTER than if you're using the default drawing mode.
  • Implement the mousePressed() event function. Inside mousePressed(), check if the mouse is over the shape and, if it is, set shapeSelected to true. There are examples showing the calculation to check if the mouse is over a rectangle in the lecture code.
  • Create the dragging effect and update the shape fill using mouseDragged(). Don't forget to check shapeSelected before moving the shape.
  • Use mouseReleased() to turn off shapeSelected and set the fill colour back to white.
