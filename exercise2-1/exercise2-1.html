Exercise 2.1

Your sketch:

Instructions

Open exercise2-1/sketch.js, add the setup() and draw() functions, and create a 600 by 600 pixel canvas. Draw a shape of your choosing in the centre of the canvas and make it take up most of the canvas space, leaving a margin of about 50 pixels around the shape.

Generate a random fill colour and save it as a global variable. To do this, declare but don't initialise a global variable called fillColour. In setup(), initialise fillColour with the following line:
fillColour = color(random(255), random(255), random(255));

The color() function is a p5.js function that is called in the same way as other colour-related functions such as fill() and background(). Use fillColour to set the fill for all shapes in your sketch by calling fill(fillColour);.

Implement the keyPressed() event function and use it to make the shape move by 5 pixels when the user presses their arrow keys or the w-a-s-d keys. To be specific:

  • When the user presses the UP arrow key or the w key, move the shape up by 5 pixels.
  • When the user presses the LEFT arrow key or the a key, move the shape left by 5 pixels.
  • When the user presses the DOWN arrow key or the s key, move the shape down by 5 pixels.
  • When the user presses the RIGHT arrow key or the d key, move the shape right by 5 pixels.

You will need to use the keyCode system variable when checking if an arrow key is pressed and the key system variable when checking if a w-a-s-d key is pressed.

When you are done, check your work using check-exercise2-1.html.

Expected output:

You will need global variables to store the current x and y coordinates of your shape. Draw the shape in draw() using your global variables to set its location.

Then, add the keyPressed() event function. The eventFunctions example from last lecture uses this function so use it as a reference if you are unsure.

Inside keyPressed(), you will need to write a conditional that checks the values of keyCode and key and moves the shape accordingly. The lineDrawingKeyboard example from lecture shows how to check key values. Use keyCode to check for the arrow keys and key to check for the letter keys.

Your conditional should have one branch for each bullet point in the list describing expected behaviour above.
